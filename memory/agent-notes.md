# Agent Notes Template

## Purpose
This file is for **agent-specific context only** — preferences, patterns, and working style that don't need to be shared with other agents.

## What Goes Here (Agent Memory)
- Personal working preferences ("I prefer to use X over Y")
- Style guide notes specific to this agent
- "Don't do this" lessons learned
- Long-term identity/personality notes

## What Goes in Main Memory (Shared Pool)
- Completed tasks and their outputs
- Handoffs to other agents
- Decisions that affect multiple agents
- Anything Jamie needs to see
